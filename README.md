# Superstore Sales Analysis

## Overview
Analyzed a **Superstore retail dataset containing 1,031 records** using **MySQL and Python** to identify sales patterns, regional performance, profitability trends, and category-wise insights through data querying and visualization.

## Business Questions Answered
-  The **West region recorded the highest sales**, indicating stronger market performance compared to other regions.
- Sales showed a **consistent peak during Q4 (October–December)**, suggesting seasonal demand trends.
- **Office Supplies had the highest order volume**, making it the most frequently purchased category.
- **Texas recorded the highest losses**, identifying it as a key area for business concern.
- Identified the **top 10 profit-generating products** to understand high-margin items.

## Technical Implementation
- Used **MySQL** to write queries for filtering, aggregation, grouping, sorting, and extracting business insights.
- Performed data analysis in **Python** using **Pandas** for handling and transformation.
- Built **visualizations using Seaborn and Matplotlib** to analyze regional sales, monthly trends, category performance, and profit patterns.

## Tools Used
- **MySQL** — business query analysis  
- **Python (Pandas, Seaborn, Matplotlib)** — data analysis & visualization  
- **Jupyter Notebook** — workflow and experimentation

## Files
- `superstore_queries.sql` — SQL queries for business analysis  
- `Superstore_visualizations.ipynb` — visualizations and trend analysis
